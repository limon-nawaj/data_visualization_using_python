Dataset Overview and Executive Summary¶

The Ford GoBike System Dataset contains information about bike-sharing rides, including details such as ride durations, user types, user ages, genders, and more. Our analysis aims to support the following key insights:

  1. User Behavior and Preferences: By exploring the relationships between user types, ages, genders, and ride durations, we aim to understand the distinct patterns that different user segments exhibit.

  2. Usage Patterns: Through visualizations, we will uncover how user types and ages influence ride durations and the distribution of rides.

  3. User Characteristics: By investigating the intersections of user types, ages, and genders, we aim to identify any trends that could help optimize service offerings.

  4. Gender Composition: We will analyze how user types are distributed across genders, shedding light on the diversity within the user base.

Let's begin our journey through the data and uncover these insights through compelling visualizations and thoughtful analysis.

plt.figure(figsize=(10, 6))
sns.violinplot(data=df, x='user_type', y='member_age')
plt.xlabel('User Type')
plt.ylabel('Age')
plt.title('User Age Distribution by User Type')
plt.show()

Ride Duration by User Type and Gender¶

This clustered bar chart shows the distribution of ride durations based on user type and gender. It provides insights into how ride durations vary among different user types and genders.

In [6]:
plt.figure(figsize=(12, 6))
sns.barplot(data=df, x='user_type', y='duration_min', hue='member_gender', ci=None)

plt.ylabel('Ride Duration (minutes)')
plt.title('Ride Duration by User Type and Gender')
plt.legend(title='Gender')
plt.show()

Heatmap of User Age and Ride Duration¶

This heatmap showcases the relationship between user age and ride duration. It provides a visual representation of where ride durations are concentrated for different age groups.

In [7]:
plt.figure(figsize=(10, 6))
heatmap_data = df.groupby(['member_age', 'duration_min']).size().unstack()
sns.heatmap(data=heatmap_data, cmap='YlGnBu', cbar_kws={'label': 'Frequency'})
plt.xlabel('Ride Duration (minutes)')
plt.ylabel('User Age')
plt.title('Heatmap of User Age and Ride Duration')
plt.show()
